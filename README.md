# VoiceRecorder

A simple iPhone voice recorder for short city-prompt audio clips.

## Features

- Large record button optimized for one-handed use on iPhone 12.
- Top-half city question prompt while the bottom half shows start, countdown, or stop controls.
- Configurable recording limit with options from 20 seconds through 5 minutes; default is 45 seconds.
- Editable city questions with random or sequential prompt selection for each recording.
- Timestamped `.m4a` recordings listed newest-first.
- Export selected recordings or export all recordings from the recordings list.

## Development

Open `VoiceRecorder.xcodeproj` in Xcode 16 or newer and run the `VoiceRecorder` target on an iPhone 12 simulator or device.
